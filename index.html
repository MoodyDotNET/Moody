moodydotnet.github.io

Lấy project về máy:

git clone https://github.com/MoodyDotNET/MoodyDotNET.github.io

Cài đặt dependency

cd MoodyDotNET.github.io
npm install

Chạy chuơng trình

dotnet run

Mở localhost:5000 trên trình duyệt để xem kết quả

Up phần của mình lên

git add .
git commit -m “nội dung đã làm”
git push

Lấy phần của mọi nguời về

git pull

Tasks trong Visual Studio Code

Phát có tạo sẵn 3 task để chạy project, để chạy task gõ Ctrl+Shift+B

Run in production mode

Này là câu lệnh “dotnet run” bình thuờng

Run in development mode (Linux)

Này là lệnh chạy dotnet ở chết độ dev cho Linux

Run in development mode (Windows)

Này là lệnh chạy dotnet ở chết độ dev cho Windows

Ở chế độ dev, các file khi save sẽ đuợc tự động compile và browser sẽ tự động load những thay đổi, chúng ta chỉ cần chạy task Development 1 lần và code thôi không cần phải chạy lệnh gì nữa hết!
